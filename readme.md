# Interactive Birthday Surprise Blueprint

A simple, multi-scene interactive web application designed as a birthday surprise.

## Features

- **Countdown Timer:** Starts the surprise.
- **Scene Management:** Transitions through different steps (Cake, Balloons, Moments, Message, Gift).
- **Interactive Elements:** Includes a button to "decorate" the cake and balloons to "pop" to reveal a hidden message.
- **Themed Design:** Uses a dark background (`#0d011f`) and pink/purple highlights (`#ff69b4`, `#8a2be2`).
